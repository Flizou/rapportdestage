Activité

Pendant mon 2ème Jour de stage, on m’a expliqué comment modifier la taille, la couleur et la police de ton texte sur ton site Web.Dans un code CSS vous allez avoir 3 éléments importants et 1 notion à retenir. Ces éléments sont : • Le sélecteur, • La propriété • La valeur La propriété couplée à une valeur induit la notion de règle (ou déclaration) qui détermine la modification apportée à un élément HTML déterminé par un sélecteur.
